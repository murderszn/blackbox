# Tool · Node Server

Static file server + API router for local BLACKBOX.

- **Entry:** `server.js`
- **Port:** `8080`
- **Cmd:** `npm start` / `npm run dev`
- **Requires:** Node ≥ 18

Serves HTML/CSS/JS/assets and mounts:

- `/api/pollinations-text`
- `/api/ai-analysis`
- `/api/signup`
