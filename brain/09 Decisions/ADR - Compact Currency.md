# ADR · Compact Currency

## Context

Quadruple-digit values overflowed metric faces, pie keys, and ledger cells.

## Decision

At 4+ digits, display K; at millions, M. Editable tiles show compact when idle, raw on focus.

## Consequences

- Dense dashboards stay legible  
- Users must focus to edit full income/amounts  
- AI plan text can use compact for consistency
