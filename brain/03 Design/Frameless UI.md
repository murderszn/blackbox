# Frameless UI

Strip stacked containers that read as “cards on cards.”

## Apply to

- Planning / purchase / viability metric washes
- Pie key rows (no boxed chrome)
- App phone media + atmosphere cells
- App Store / Play badges
- X / LinkedIn social row
- Journey spine pill (lighter, no heavy border stack)
