# Typography

| Role | Family | Use |
|------|--------|-----|
| Display | Cormorant Garamond | Titles, big amounts |
| UI | Syne | Labels, body UI |
| Data | JetBrains Mono | Metrics, tables, kickers, rail labels |

## Density

- Chart titles: calm, not billboard
- App stage: quieter than hero ([[Stage - App]])
- Planning faces: scale by `data-digits`; compact K overlay at 4+ digits
