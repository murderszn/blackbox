# Skill · Visual Consistency Pass

Walk the full film once and fix only consistency:

- [ ] Same gold hairlines / no random borders  
- [ ] Same radius language (prefer 0–3px frameless)  
- [ ] Display / UI / data fonts in correct roles  
- [ ] Button shapes match (stores, social, signup)  
- [ ] Card/metric padding scale similar across stages  
- [ ] Numbers use [[Compact Currency]] everywhere dense  
- [ ] Left rail not colliding with content
