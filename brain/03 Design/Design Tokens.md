# Design Tokens

Obsidian Atelier — deep black vault + champagne gold.

| Token | Role (approx) |
|-------|----------------|
| `--gold` / `--gold-soft` / `--gold-deep` | Accents, rails, highlights |
| `--text` / `--text-secondary` / `--text-tertiary` / `--text-muted` | Hierarchy |
| `--border` / `--border-light` | Hairlines only |
| `--font-display` | Cormorant Garamond (editorial) |
| `--font-ui` | Syne |
| `--font-data` | JetBrains Mono |
| `--ease-out-expo` | Motion |

## Rules of thumb

- Prefer **wash backgrounds** over nested cards
- Borders: hairline gold alpha or none ([[Frameless UI]])
- Data numbers: mono + tabular + [[Compact Currency]]
