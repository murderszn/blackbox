# Skill · Stage Full Viewport

Each `.stage-panel` should feel full-screen:

- `min-height: 100dvh`  
- Content flex-centered or stretched intentionally  
- Avoid `overflow: hidden` that clips forms  
- Snap: **proximity** + `scroll-snap-stop: normal`  

See [[ADR - Proximity Scroll]] · [[Journey Film]]
