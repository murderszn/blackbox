# Skill · Critique Cycle

Awwwards-oriented loop (3+ passes when raising the bar):

1. **Capture** screenshots of target stages  
2. **List ≥5 problems** (type, spacing, hierarchy, chrome, motion)  
3. **Fix** only those problems  
4. **Recapture**  
5. Repeat  

Store shots under repo root `qa-*.png` or `brain/assets/`.  
Pairs with [[Skill - Playwright Visual QA]].
